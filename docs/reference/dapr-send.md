# dapr send

## Description

Invoke a Dapr app with an optional payload

## Usage

```bash
dapr send [flags]
```

## Flags

| Name | Environment Variable | Default | Description
| --- | --- | --- | --- |
| `--app-id`, `-a` | | | The app ID to invoke |
| `--help`, `-h` | | | Help for send |
| `--method`, `-m` | | | The method to invoke |
| `--payload`, `-p` | | | A JSON payload |
