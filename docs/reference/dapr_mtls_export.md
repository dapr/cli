## dapr mtls export

Export the root CA, issuer cert and key from Kubernetes to local files

### Synopsis

Export the root CA, issuer cert and key from Kubernetes to local files

```
dapr mtls export [flags]
```

### Options

```
  -h, --help         help for export
  -o, --out string   Output directory path to save the certs (default ".")
```
