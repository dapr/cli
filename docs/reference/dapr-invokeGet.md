# dapr invokeGet

## Description

Issue HTTP GET to Dapr app

## Usage

```bash
dapr invokeGet [flags]
```

## Flags

| Name | Environment Variable | Default | Description
| --- | --- | --- | --- |
| `--app-id`, `-a` | | | The app ID to invoke |
| `--help`, `-h` | | | Help for invokeGet |
| `--method`, `-m` | | | The method to invoke |
