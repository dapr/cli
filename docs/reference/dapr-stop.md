# dapr stop

## Description

Stops running Dapr instances and their associated apps

## Usage

```bash
dapr stop <appID1> <appID2> [flags]
```

## Flags

| Name | Environment Variable | Default | Description
| --- | --- | --- | --- |
| `--app-id` | | | The app ID to stop (standalong mode) |
| `--help`, `-h` | | | Help for stop |
