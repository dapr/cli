# dapr stop

## Description

Stops a running Dapr instance and its associated app

## Usage

```bash
dapr stop [flags]
```

## Flags

| Name | Environment Variable | Default | Description
| --- | --- | --- | --- |
| `--app-id` | | | The app ID to stop (standalong mode) |
| `--help`, `-h` | | | Help for stop |
